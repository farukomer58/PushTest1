FYS Voorbeeld

Dit project bevat een voorbeeld structuur die je kunt aanhouden voor je eigen FYS project.

Plaats alles wat je op je website beschikbaar wil hebben (HTML, JavaScript, CSS, afbeeldingen, videos, enzovoorts) onder de www-map. Hierbij is het gebruikelijk om alle niet-HTML bestanden netjes te verdelen onder de www/assets-map!

Wil je gebruik maken van externe bronnen, zoals jQuery of Bootstrap? Plaats die dan in een map onder www/assets/vendors, dit houdt jouw code gescheiden van externe bronnen.

Dit project bevat ook een .gitignore-bestand, hiermee voorkom je dat bepaalde bestanden (perongeluk) op Git worden ingechecked. Denk bijvoorbeeld aan .DS_STORE-bestanden die elke Mac-gebruiker aanmaakt, of aan de workspace.xml van IntelliJ die voor alle gebruikers weer anders is.

Klik op onderstaande knop om plaatjes te tonen en te verbergen!
